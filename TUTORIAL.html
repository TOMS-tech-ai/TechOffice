إنشاء حساب السوبر أدمن
تسجيل المستخدم ومنحه صلاحيات الإدارة الكاملة
🔐 خطوة 1 — إنشاء المستخدم في Supabase Auth
  • 1️⃣
    افتح Authentication من القائمة الجانبية في Supabase
  • 2️⃣
    اضغط + Add user → Create new user
  • 3️⃣
    أدخل إيميل وباسورد قوي — مثلاً: superadmin@techoffice.com
  • 4️⃣
    ضع علامة على Auto Confirm User لتفعيل الحساب فوراً
⚡ خطوة 2 — منح صلاحية السوبر أدمن (SQL)
SQL EDITOR
-- استبدل الإيميل بإيميلك الفعلي SELECT public.set_superadmin_role('superadmin@techoffice.com'); -- ✅ النتيجة المتوقعة: set_superadmin_role -- (يعني نجح التنفيذ)
✅
بعد تنفيذ الدالة، سيظهر هذا المستخدم في جدول superadmins وسيحمل user_metadata.role = "superadmin" في Supabase Auth
💡
يمكنك إنشاء أكثر من سوبر أدمن بتكرار نفس الدالة مع إيميلات مختلفة

ربط المشروع بـ Supabase عبر مفاتيح API
📋 أين تجد مفاتيح Supabase؟
  • 🔧
    في Supabase → افتح Project Settings (⚙️ أسفل القائمة الجانبية)
  • 🔌
    اضغط على API من القائمة الفرعية
  • 📋
    ستجد: Project URL و anon public key — هذان هما المطلوبان
📝 إنشاء ملف .env.local
TERMINAL
cp .env.example .env.local
.env.local
VITE_SUPABASE_URL=https://xxxxxxxxxxx.supabase.co VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9... # اختياري — بوابة الدفع Paymob VITE_PAYMOB_API_KEY=ZXlKaGJHY2lPaUpJVXpVeE1pSXNJblI1... VITE_PAYMOB_INTEGRATION_ID=123456
⚠️
مهم: لا تشارك ملف .env.local مع أحد ولا ترفعه على GitHub. المشروع يحتوي على .gitignore يمنع رفعه تلقائياً.
